♟️ Projeto Xadrez em C#

📌 Objetivo

Desenvolver um jogo de xadrez utilizando a linguagem C#, aplicando conceitos de programação orientada a objetos, lógica de programação e estruturas de dados.

🎯 Objetivos Específicos
Criar as classes iniciais que representam o tabuleiro e suas posições.
Implementar as peças do jogo (Rei, Rainha, Torre, Bispo, Cavalo e Peão).
Desenvolver as regras de movimentação de cada peça.
Validar jogadas conforme as regras oficiais do xadrez.
Exibir o tabuleiro no console de forma clara e organizada.

🧠 Conceitos Aplicados
Programação Orientada a Objetos (POO): criação de classes, objetos, encapsulamento, herança e polimorfismo.
Estruturas de repetição e decisão: loops e condicionais para validar movimentos e turnos.
Manipulação de matrizes: representação do tabuleiro e controle das posições das peças.
